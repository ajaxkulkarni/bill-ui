1. Pay Per Bill Mobile Application

Why Pay Per Bill? 

Our app is a one stop interface for newspaper vendors and customers. It makes life easy when you want to pause delivery, pay bills online or subscribe to a new newspaper. 

Debit Card, Credit Card, Net Banking, BHIM, UPI these payment methods are available with this application, customer can pay for their vendor with any of this payment method. The main objective of developing this application is to track daily newspaper quantity, daily orders, customer/vendor holidays, sms/email/WhatsApp alerts for payments from vendor to customer.

Online bill tracking and payment system is very important and helpful to reduce the petrol cost, travel cost and human efforts.

How to start?

This application is very easy and user friendly to use, to use this application android phone, internet connection is necessary and it should be available with good strength of network. This documentation will help you to use this application and also important to know more about it. Every vendor should read this document before using PayPerBill app on their mobile phone.

In this application vendor (vikreta) need to update their personal data like his full name, email, contact, PAN Id, Aadhar No, Business Name and Business license number after getting logged into this app with the photo copy of PAN id and Adhar No.

After updating and submitting personal information through this application PayPerBill will verify all those documents and then they will approve the vendor to use this application. (It will take approximately 1-2 working days to approve the vendor.)

Our PayPerBill application is available on Play Store after downloading that application vendor need to log in with own valid mobile number which will be used for login with OTP.

2.LOGIN  


1. Enter a valid mobile number

4. If OTP is not received via SMS, click on RESEND to get OTP again.
